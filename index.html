Do prompt ao commit:
criando jogos no navegador com Claude Code

Como um app no meu Mac + uma conta no GitHub viraram uma fábrica pessoal de protótipos jogáveis — sem código complicado, sem fricção.

  • 💻 Claude Code instalado no Mac — funciona como um assistente de programação rodando direto no meu computador. Conectado ao meu GitHub, ele cria, edita e salva os arquivos do projeto pra mim. Sem editor de código, sem ambiente complicado pra configurar.
  • 🎮 Os jogos rodam no navegador — feitos em HTML + JavaScript (a "linguagem da web"). Quem joga só clica no link — sem download, sem instalação, funciona no celular ou no PC.
  • ⚡ Eu converso, ele programa — descrevo a ideia em português ("quero uma torre que congela inimigos") → o Claude escreve o código → testo na hora no navegador → peço ajustes → ele salva tudo no GitHub automaticamente.
  • 🔁 Da ideia ao protótipo em minutos — ciclo curtíssimo entre "quero X" e ver rodando na tela. Bugs corrigidos no chat, sem sair do contexto.
  • 🧠 Meu papel: diretor criativo — eu cuido do conceito, mecânica, balanceamento e feel do jogo (o que um game designer faz). O Claude cuida da parte técnica.

4 protótipos jogáveis,
um fim de semana de experimentação

Cada jogo nasceu de uma conversa. Todos rodam no navegador, todos com histórico salvo no GitHub.

  • 🏰 Dark Tower · Tower Defense — jogo de defender o castelo: você posiciona torres num tabuleiro pra impedir os inimigos de passar. Tem ondas que ficam mais difíceis, torres que melhoram com upgrades, economia de ouro e inimigos que sabem achar o melhor caminho até você.
  • 💀 Doomzinho · FPS 3D — homenagem ao clássico Doom dos anos 90. FPS = jogo de tiro em primeira pessoa (você vê pelos olhos do personagem). 3D rodando no navegador, mira pelo mouse, inimigos que perseguem.
  • 🥽 Doomzinho VR · Realidade Virtual — o mesmo jogo, mas dentro do óculos VR (Meta Quest). O navegador hoje suporta VR nativamente, sem precisar baixar app — é só abrir o link pelo headset e jogar com hand-tracking (controle pelas mãos).
  • 🚀 Star Fox-like · Nave em trilho — homenagem aos jogos de nave do Super Nintendo. Você voa numa rota fixa atirando em tudo que aparece, com aquele visual "blocudo" retrô (low-poly) que define a estética da era 16/32-bit.

O que mais me marcou: com Claude Code rodando no Mac e o GitHub fazendo a publicação, a distância entre "ter uma ideia de jogo" e "ter o jogo rodando pra mostrar pra alguém" praticamente sumiu. O gargalo deixa de ser técnico — passa a ser criativo.
